# Roadmap

## Phase 1

- Local scanner.
- Metadata reader.
- SQLite storage.
- Genre normalization.
- Review queue foundation.
- CSV export for scanned records and proposed tags.
- Manual correction import with correction history.
- Experimental BPM detection stored in SQLite.
- Experimental beat timestamp storage and cue point proposals.

## Phase 2

- Safer metadata write-back with backups.

## Phase 3

- AcoustID or MusicBrainz lookup.
- External metadata confidence scoring.

## Phase 4

- Improved BPM detection.
- Improved beat-grid detection.
- Cue point proposal engine.

## Phase 5

- Engine DJ, Rekordbox, and MIXO export adapters.
